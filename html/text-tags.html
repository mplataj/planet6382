Use Accurate HTML Tags

There are many different HTML elements that can be used to add visible effects to text, or to give invisible contextual information for screen readers.

Often there are choices about which element tag to use, or they are entirely optional. Below are the top reasons to consider using the most accurate HTML element tags possible.

  1. Using accurate inline HTML tags for text increases the accessibilty of a webpage, because they give more contextual information for screen readers.
  2. Using the correct and most specific HTML tags increases the accuracy of search engine optimization.
  3. Certain HTML tags are important in academic, government, or other institutions where scholarship and documentation are critical.

HTML vs CSS for Text Styling

It is best practice to do general text styling for a webpage, such as font sizes or colors, using a CSS stylesheet.

However, for many common text effects that are used in single-case incidences, such as italicising a word in a sentence, it is most effcient to use inline HTML element tags.

Element Tags for Text and Context

Below is a scrollable list of some common HTML element tags that are used for visual text effects and/or invisible context:

  • Bold <b> </b>

    • Bolded text looks the same as using the <strong> </strong> element tags. However, bold tags are used for presentational style, while the strong tags are used to indicate greater semantic importance.

  • Line Break <br>

    • Using the break tag will create a single line break
    in the text. NOTE: the line break element does not need a closing tag.

  • Delete <del> </del>

    • Delete looks the same as the <s>strikethrough</s> element tags, but is used for text editors to show their editing cuts.

  • Emphasis <em> </em>

    • In most browsers, text with emphasis looks the same as italicized text. However, emphasis tags convey to the screen reader that the indicated text is given more meaning or importance than the surrounding words.

  • Insert <ins> </ins>

    • The insert element looks just like underlined text. However, insert tags are only used for text editors to indicate where a word insertion should take place.

  • Italics <i> </i>

    • Italicized text looks the same as using the citation or emphasis element tags. However, the italics tags should be used when only a visual effect is intended. For titles of cited works, such as books or songs, it is best practice to use the <cite></cite> element tags. And if contextual emphasis is intended, then it is best to use the <em></em> element tags for screen readers.

  • Strikethrough <s> </s>

    • Strikethrough text looks the same as the <del>delete</del> tags, but is used for intentional visual effect rather than editing purposes.
