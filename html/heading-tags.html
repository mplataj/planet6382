Heading Tags

HTML heading tags are used to distinguish specific text on a webpage as a heading that will quickly stand out from the surrounding body text to help readers better navigate the information on the page.

There are six heading tags which have bolded font weights to increase their visibility, and a range of default sizes starting from largest (#1) to smallest (#6) to designate their significance level.

These are the six standardized HTML heading tags:

<h1> </h1>
<h2> </h2>
<h3> </h3>
<h4> </h4>
<h5> </h5>
<h6> </h6>

NOTE:  The heading element always needs both an opening and closing tag.


Heading Tags for SEO

Heading tags not only make a webpage easier to navigate for readers, but using heading tags effectively is also important for search engine optimation.
