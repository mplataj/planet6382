How to Examine the Source Code of Any Web Page

To closely analyze any cyberspace artifact you can view it's underlying source code by right-clicking your mouse button, and then choosing "View page source" or "Inspect" from the menu that appears.

The difference between the two options is that "View page source" will open a new web page to view the source code of the whole webpage; while "Inspect" will target a specific webpage element, and will split the current pageview rather than opening a new window. Additionally, 'Inspect' opens developer tools for code editing and troubleshooting.
