HTML Iframe Example

Iframes are used to create a frame that displays interactive content from another webpage. The term iframe stands for 'inline frame.'

Below is an example of code for an iframe that displays the West Texas A&M homepage:

<iframe
width="450"
height ="350"
src="https://www.wtamu.edu/">
</iframe>

Iframes for Displaying Maps

A common use for iframes is to display an interactive map on a website. Google Maps is the most common source. You can copy and paste the exact iframe code needed directly from the Google Maps website.

To get the iframe code for a Google map follow these steps:

  1. Visit Google Maps and search for the address you want to display.
  2. Click on the "Share" option.
  3. On the pop-up that displays, click the second tab option: "Embed a map".
  4. Select the size of the iframe using the drop down menu on the left.
  5. Copy the code and paste it onto your webpage.
