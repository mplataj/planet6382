The HTML Video Element

There are multiple ways to add video to your website. The most straightforward way is to upload your video file to your website and use the HTML <video> </video> tags.

The video element has many attributes which give you control over how it displays and plays. Below is a list of the most common video attributes:

  • Src= " "

    • Src is the only required attribute of the video element, and it provides the file path of the video source. Src needs an equals sign followed by the file path location within quotes.

  • Width= " " and Height= " "

    • Width and height are two other key attributes that define the size of the video displayed. Width and height both need an equals sign followed by a size value within quotes.

  • Poster= " "

    • The poster attribute lets you set a thumbnail image to display until the video starts to play. Poster needs an equals sign followed by the image location within quotes.

  • Controls

    • The controls attribute tells the browser to add it's default controls. The controls attribute does not need to be given a value.

  • Autoplay

    • The autoplay attribute tells the browser to automatically begin playing the video as soon as it loads. The autoplay attribute does not need to be given a value. However, you can add the 'muted' attribute after autoplay to make it play silently.

  • Loop

    • The loop attribute tells the browser to continually restart the video when it reaches the end, until the user presses stop. The loop attribute does not need to be given a value.

HTML Video Example

Below is an example of HTML code for a video:

<video
src="../media/telephone1946.mp4
poster="images/dial-telephone.jpg"
width="400"
height="300"
controls
autoplay muted
loop
>

<p>Text between the video tags displays if your video doesn't load.</p>

</video>
