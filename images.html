HTML Image Element

The image element allows you to display images on a webpage. The image element opening tag is: <img>. The image element does not have a closing tag.

Below is a list of the most common image attributes:

  • Src=" "

    • Src is the main required attribute of the image element, and it provides the file path of the image source. Src needs an equals sign followed by the file path location within quotes.

  • Alt=" "

    • The image element requires an 'alt' attribute for accessibility, which gives an alternate text description of the image for screen readers. It is also the text that displays if the image fails to load.

  • Width=" " and Height=" "

    • Width and height are two other key attributes that define the size of the image displayed. Width and height both need an equals sign followed by a pixel size value within quotes.

  • Title=" "

    • You can use the title attribute to set text that will display when the user's cursor hovers over the image.


HTML Figure Elements

  • <figure> </figure>
  • <figcaption> </figcaption>

Figure and figure caption are not attributes of the image element. However, they are elements very commonly used along with it.

The <figcaption> tags are the only way to directly associate an image caption in HTML. And to use the figure caption element properly, the image and its caption should both be contained within the <figure> element tags.

If you do not want to associate an image caption with your image, then there is no need to use the figure elements.

Image Example

Below is an example of HTML code for an image:

<figure>

<img
alt="Planet Earth viewed from space"
title="Planet Earth"
width="300"
height ="300"
src="../images/earth.jpg"
>

<figcaption>Home Sweet Home </figcaption>

</figure>
