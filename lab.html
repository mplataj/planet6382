How to Examine the Source Code of Any Web Page

To closely analyze any cyberspace artifact you can view it's underlying source code by right-clicking your mouse button, and then choosing "View page source" or "Inspect" from the menu that appears.

View Page Source

View page source will open a new window or tab to view the source code of the whole webpage.

Inspect

Inspect will target a specific webpage element, and will split the current pageview rather than opening a new window. Additionally, 'Inspect' opens developer tools for code editing and troubleshooting.
