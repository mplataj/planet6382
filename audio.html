The HTML Audio Element

There are multiple ways to add audio to your website. The most straightforward way is to upload your audio file to your website and use the HTML <audio> </audio> tags.

The audio element has many attributes which give you control over how it displays and plays. Below is a list of the most common audio attributes:

  • Src= " "

    • Src is the only required attribute of the audio element, and it provides the file path of the audio source. Src needs an equals sign followed by the file path location within quotes.

  • Controls

    • The controls attribute tells the browser to add it's default controls. The controls attribute does not need to be given a value. The look of the control will vary depending on the browser used. It is necessary to use the controls attribute unless you are adding your own customized javascript controls.

  • Autoplay

    • The autoplay attribute tells the browser to automatically begin playing the audio as soon as it loads. The autoplay attribute does not need to be given a value.

  • Loop

    • The loop attribute tells the browser to continually restart the audio when it reaches the end, until the user presses stop. The loop attribute does not need to be given a value.

HTML Audio Example

Below is an example of HTML code for playing audio:

<audio
src="../media/inconclusions.wav"
controls
loop
>

<p>Text between the audio tags displays if your audio doesn't load.</p>

</audio>
