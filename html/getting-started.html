What is a HTML element tag?

HTML element tags are the opening and closing sequence of characters for each HTML element that define what command is being communicated and carried out. The element tag creates a container which defines where the command instructions begin and end.

Every HTML element has an opening tag which begins with the "<" symbol and ends with the ">" symbol. Most, but not all, HTML elements also require a closing tag.

You can learn more about HTML element tags at W3schools.com. And to see an complete list of all the HTML elements click here.


Getting Started

There are certain HTML elements which every webpage needs to have, and other elements which are strongly reccommended.

Below is a list of the essential elements needed to start your HTML webpage:

  • <!doctype html>

    • Every HTML webpage is required to have a document type element which tells the browser the webpage is written in HTML.

  • <html lang="en">

    • It is highly reccommended to also tell the browser what human language is being primarily used on the webpage. This helps search engines and browsers.

  • <head> </head>

    • After the doctype and language type, it is best practice to begin the <head> element. Not to be confused witht the heading tags, the head element is used to record information about the webpage that is not viewed on the website itself. This includes the document title, stylesheet reference, and any other meta data.

  • <title> </title>

    • Every HTML webpage is required to have a title. The title name will appear on the tab in the browser window. The title tags are included somewhere within the <head> tags.

  • <link href="../styles.css" rel="stylesheet" type="text/css">

    • If your webpage has a separate CSS file, which most do, then you need to link the stylesheet somewhere within the head element tags.

  • <style> </style>

    • If you are including your CSS styling within the same webpage, rather than a separate file, then the CSS code will be contained within the <style> </style> tags, which are contained somewhere within the head element section.

  • <meta>

    • There are many types of meta data that can be included in the head section to give behind-the-scenes reference information that helps browsers, search engines, screen readers, and other interested parties best use your webpage. NOTE: Meta elements do not have closing tags.

    See a list of important meta attributes in the next window.

Meta Data Tags

There are many types of meta data that can be included in the head section to give behind-the-scenes reference information that helps browsers, search engines, screen readers, and other interesrted parties best use your webpage.

This is the meta data tag: <meta>

NOTE: Meta elements do not have closing tags.

Some of the most commonly used meta attributes are listed below:

  • <meta charset="UTF-8">

    • This tells the browser what character set is being used.

  • <meta name="viewport" content="width=device-width, initial-scale=1.0">

    • This adjusts your webpage dimensions so that it will look best on different device viewports.

  • <meta name="author" content="Web Author Name">

    • This cites the author of the webpage.

  • <meta name="description" content="Add a description of your webpage.">

    • Here you can give a description of your webpage contents, of up to 155 characters in length, that helps search engines.

  • <meta name="keywords" content="keyword1, keyword2, keyword3">

    • Here you can choose keywords for search engine optimization.

Webpage Head Example

Below is an example of standard practice HTML code commonly found at the top of a webpage:

<!doctype html>
<html lang="en">
<head>

<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">

<title>Planet 6382</title>
<meta name="author" content="Author Name">
<meta name="description" content="HTML and CSS reference guide">
<meta name="keywords" content="HTML, CSS, web design">

<link href="../styles.css" rel="stylesheet" type="text/css">

</head>

Body Elements Example

After the head section comes the <body> </body> tags. The body of the webpage includes all of the visible content. It is usually divided into three standard parts: the navigation, main, and footer sections.

After the body tag closes, the opening HTML element tag also gets closed, ending the webpage. See an example below:

<body>

<nav>
navigation links go here
</nav>

<main>
page content goes here
</main>

<footer>
footer information goes here
</footer>

</body>
</html>
